A. 어플 개발 방법

네이티브 앱 (안드로이드 or IOS 에서만 작동)
- 안드로이드 : 코들린, 자바로 안드로이드 스튜디오 사용
- IOS : 쉬프트, 오브젝티브-C 로 XCODE 사용
* 네이티브 앱을 만들어야 성능이 가장 뛰어나지만 각각을 학습해야하는 부담.
하이브리드 앱 (하나의 언어로 각각에서 작동)
- React Native 라는 프레임 워크를 사용해 Java script 언어 하나로 개발 가능
- 구글 Flutter 를 사용해 Dart 라는 언어로 개발 가능
웹뷰 (가장 쉬운 방법)
- 앱 화면에 내가 만든 웹 URL 을 띄워 돌리는 방식
앱 인벤터(교육용 앱 제작 도구)
앱 인벤터(교육용 앱 제작 도구)
유니티
게임어플을 만들 경우에는 유니티를 사용해 java script 나 C#을 이용할수도 있다

B. 어플 구조

프론트 앤드
- HTML(마크업 언어. 밑그림), CSS (물감), Java script(움직임 만듬) 사용
- 자바스크립트부터 진정한 언어이기 때문에 양이 방대하다. 꼭 필요한 부분만 짚고 넘어간다.
- 구글이나 네이버에서 제공하는 API를 잘 활용하면 백앤드 없이 개발도 가능하다.
백 앤드
- 자바스크립트, 파이썬, 루비 등을 사용해 가장 기본적인 개념인 변수, 조건문, 반복문, 함수 4가지 개념만 학습하는 것을 추천
- 전에는 깊이 알아야 했지만 이제는 웹 프레임 워크에서 그런 역할을 해줌
- 언어별로 용도별로 다양한 프레임 워크가 존재한다. 자바스크립트-스프링, 파이썬-장고, 루비-레일즈. 초보자는 루비 추천
